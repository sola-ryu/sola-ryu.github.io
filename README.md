# Sola — Digital Presence

This is the repository for [sola-ryu.github.io](https://sola-ryu.github.io), a quiet personal site built with Astro. It holds my blog posts, project notes, and scattered thoughts.

## What's Here

- **Blog** — Posts I actually wrote, not template filler.
- **Projects** — Documentation for things I've built or maintained.
- **About** — Who I am, why this site exists.

The site is a fork of [astro-navfolio](https://github.com/dodolalorc/astro-navfolio). The original repo has the full module documentation, theme guides, and setup instructions if you want to see how it works under the hood. I've stripped out the template detritus and kept what's mine.

## Running Locally

```sh
bun install
bun run dev
bun run build
bun run preview
```

## Config

Everything sits in `src/config/site.toml` — title, profile, navigation, theme palette, search, comments. No component edits needed for personalization.
